# 五子棋游戏 (Gomoku)

这是一个用Python实现的五子棋游戏，包含完整的棋盘、落子、胜负判断等功能。

## 功能特点

- 15x15 标准棋盘
- 黑棋和白棋轮流落子
- 自动检测胜负（五子连珠）
- 棋盘可视化显示
- 有效的输入验证

## 文件说明

- `gomoku.py` - 完整的交互式五子棋游戏
- `gomoku_demo.py` - 演示版本，展示游戏核心功能

## 如何运行

要运行交互式游戏：
```bash
python gomoku.py
```

要查看演示：
```bash
python gomoku_demo.py
```

## 游戏规则

1. 黑棋先行
2. 玩家轮流在棋盘上放置自己的棋子
3. 首先在横、竖或斜方向上连成五子的一方获胜
