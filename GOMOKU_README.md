# 五子棋游戏

这个项目包含两个版本的五子棋游戏：

## Python 版本 (gomoku.py)
- 这是一个命令行版本的五子棋游戏
- 使用 NumPy 库来管理游戏棋盘
- 两个玩家轮流在 15x15 的棋盘上放置棋子
- 先连成五子的一方获胜

### 如何运行 Python 版本:
1. 确保已安装 Python 3 和 NumPy
2. 在终端中运行: `python gomoku.py`
3. 按照提示输入行号和列号 (0-14) 来放置棋子

## HTML 版本 (gomoku.html)
- 这是一个图形界面版本的五子棋游戏
- 可以在浏览器中直接运行
- 黑白两色棋子，直观易懂
- 包含获胜棋子高亮显示功能

### 如何运行 HTML 版本:
1. 双击 `gomoku.html` 文件
2. 或在浏览器中打开该文件
3. 点击棋盘上的任意空位来放置棋子

## 游戏规则
- 黑棋先行
- 两个玩家轮流放置棋子
- 率先在横向、纵向或对角线方向上连成五子的一方获胜
