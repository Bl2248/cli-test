# 五子棋 (Five in a Row) 游戏

这是一个包含网页版和命令行版的五子棋游戏实现。

## 文件说明

- `gomoku.html` - 网页版五子棋游戏 (支持鼠标点击)
- `gomoku-cli.js` - 命令行版五子棋游戏 (支持键盘输入)
- `test-gomoku.js` - 游戏逻辑测试

## 使用方法

### 网页版 (gomoku.html)
1. 直接在浏览器中打开 `gomoku.html` 文件
2. 黑白双方轮流点击棋盘落子
3. 率先连成五子的一方获胜

### 命令行版 (gomoku-cli.js)
运行以下命令开始游戏：
```bash
node gomoku-cli.js
```

输入格式为 `行 列` (例如: `7 7` 表示中心位置)
- 黑子: ●
- 白子: ○
- 退出游戏: 输入 `quit` 或 `exit`

### 运行测试
```bash
node test-gomoku.js
```

## 游戏规则

1. 游戏在15×15的棋盘上进行
2. 黑白双方轮流落子
3. 率先在横、竖或斜方向上连成五子的一方获胜
