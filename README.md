# 五子棋游戏

这是一个网页版的五子棋游戏，使用HTML、CSS和JavaScript实现。游戏支持双人对战，具有完整的五子棋规则和良好的用户体验。

## 功能特点

- **经典五子棋玩法**：15×15标准棋盘，黑子先行
- **禁手规则**：黑子有三三禁手规则（不能同时形成两个活三）
- **动画效果**：棋子落子动画、获胜棋子脉冲动画
- **音效支持**：落子音效、获胜音效、禁手提示音效
- **交互提示**：游戏状态显示、禁手提示、消息通知
- **辅助功能**：悔棋功能、重新开始功能
- **美观界面**：现代化UI设计，毛玻璃效果，响应式布局

## 游戏规则

1. 黑子先行，双方轮流在棋盘上放置棋子
2. 率先在横、竖、斜方向连成五子者获胜
3. 黑子有禁手规则：不能形成两个活三（三三禁手）

## 使用方法

1. 在浏览器中打开 `gomoku.html` 文件
2. 黑子先行，点击棋盘上的交叉点落子
3. 交替落子直到有一方连成五子获胜

## 控制按钮

- **重新开始**：开始新游戏
- **悔棋**：撤销上一步棋

## 技术实现

- 使用HTML5、CSS3和原生JavaScript开发
- 使用CSS Grid实现棋盘布局
- 使用Canvas或DOM元素实现棋子绘制
- 使用CSS动画实现视觉效果
- 使用Web Audio API（或音频元素）实现音效

## 浏览器兼容性

支持现代浏览器（Chrome, Firefox, Safari, Edge等）
