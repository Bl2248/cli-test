# 五子棋游戏

一个基于HTML、CSS和JavaScript实现的网页版五子棋游戏。

## 功能特性

- 15×15标准棋盘
- 黑白双方轮流落子
- 智能胜负判定（横向、纵向、对角线五子连线）
- 获胜棋子高亮显示
- 重新开始游戏功能

## 如何运行

1. 使用任何HTTP服务器运行项目，例如：
   ```bash
   python3 -m http.server 8080
   ```
2. 在浏览器中打开 `http://localhost:8080/gomoku.html`

## 游戏规则

- 黑子先行
- 双方轮流在棋盘交叉点上落子
- 率先在横、竖或对角线上形成连续五子的一方获胜
